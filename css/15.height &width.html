How to set height and width to HTML Element?

set height & Width to an HTML Element CSS provides below properties:

Height

Width

To set maximum width to an HTML Element CSS provides max-width property. maximum width capacity 400px

To set minimum width to an HTML Element CSS provides min-width property. minimum width capacity 200px

To set maximum height to an HTML Element CSS provides max-height property. maximum height capacity 200px
>To set minimum height to an HTML Element CSS provides min-height property. minmum height capacity 200px
