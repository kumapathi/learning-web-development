Padding are used to create spacing around html elements.

Padding can be applied all four sides of an element: top, right, bottom, left.

Below are the CSS Properties for adding Padding for each side of the element.

Padding shorthand
padding-top
padding-right
padding-bottom
padding-left

padding-3values
padding-top is 1st one
padding-left & margin-right is 2nd one
padding-bottom is 3rd one

padding-2values
padding-top & bottom is 1st one
padding-right & lift is 2nd one

padding-1value
padding-top, right, bottom, left its totaly only one value
