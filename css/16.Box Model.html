CSS Box = margins + borders + padding + height + width of an element.
All HTML Elements can be considered as boxes.
CSS Box that wraps around every HTML Element.
CSS Box can be used as a toolkit for customizing the layout of different elements.

How to Calculate Actual Height of an HTML Element?
ex:- p{
border: 10px solid orange;
margin: 20px;
padding: 30px;
height: 440px;
}

Actual Height
= 440px(height)
+ 40px(top + bottom margin)
+ 20px(top + bottom border)
+ 60px(tpp + bottom padding)
= 245px
Actual Height = Height of an element + marign (top + bottom value) + padding (top + bottom value) + border (top + bottom value);

440px + 40px + 20px = 600px + 60px = 660px

How to Calculate Actual Width of an HTML Element?
ex:- p{
border: 5px solid red;
margin: 75px;
padding: 45px;
width: 200px;
}

Actual Width
= 200px(Width)
+ 150px(top + bottom margin)
+ 10px(top + bottom border)
+ 90px(tpp + bottom padding)
= 450px

Actual width = width of an element + marign (left + right value) + padding (left + right value) + border (left + right value);

500px + 50px + 100px + 30px = 680px
