HTML Elements are categorized in two types:

1. Block level element.

A Block level element always start on a new line.

A Block level element always takes up full width available.

CSS property can be used convert inline element to block level.



2. Inline Element. An Inline element does not start ona new line. An Inline element always takes up only necessary width. An Inline element always takes up only necessary width. CSS property can be used convert block level element to inline level.
