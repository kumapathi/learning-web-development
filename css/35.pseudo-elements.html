A CSS Pseudo-Elements is used to style specified parts of an element.
Example: Style the first letter or first line of an element.
Insert content before or after the eontent of an element etc.
Types of CSS pseudo elements:
::before, ::after, ::first-letter, ::first-line, ::marker, ::placeholder, ::selection, ::backdrop, ::file-selector-botton, ::cue, ::part( ), ::slotted( )
