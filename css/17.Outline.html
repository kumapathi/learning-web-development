An outline is a line drawn outside the elements border:

Below are the CSS Outline Properties:

Outline-style: solid, dashed, dotted, double, groove, ridge, inset, outset, hidden, none

Outline-color: color name, HEX, RGB, HSL



Outline-width: thin(typically 1px), medium(typically 3px), thick(typically 5px), A specific size(in px, cm, etc.)

Outline: Shorthand property:- outline-width + outline-style + outline-color (outline:2px inset red)

Outline-offset: Outline-Offset is a property adds space between an outline and border of an element
