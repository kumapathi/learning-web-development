Margins are used to create spacing around html elements.

Margin can be applied all four sides of an element: top, right, bottom, left.

Below are the CSS Properties for adding margin for each side of the element.

margin shorthand

margin-top

margin-right

margin-bottom

margin-left

margin-3values

margin-top is 1st one

margin-left & margin-right is 2nd one

margin-bottom is 3rd one

Margins-2values

margin-top & bottom is 1st one

margin-right & lift is 2nd one

Margin-1value
margin-top, right, bottom, left its totaly only one value
